❓ Help & Support

1 Item Management Page

  • Fill in item information (ID must be unique), click "Add Item" to add a new item
  • Click the "Delete" button next to an item to remove it

2 Search Page

  • Enter an item name and click "Search" to find matching items
  • Supports fuzzy search, partial names can be matched

3 FAQ

  • Failed to add? Please check if the ID is duplicated or if required fields are empty
  • Page not navigating? Please check if the routing configuration is correct
  • Data lost? This system stores data locally only, clearing browser cache will result in data loss

💡 Need More Help?

If you encounter any other issues, please contact the system administrator for assistance.
